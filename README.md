# Currency Converter

## Project Overview
This project is a **Currency Converter** web application that demonstrates **core JavaScript concepts** and **API integration**. It allows users to convert any amount between the world's currencies in real-time, providing an interactive and user-friendly interface.

## Features
- Convert amounts between any two currencies.
- Real-time exchange rates fetched from a currency API.
- Interactive and responsive GUI using HTML and CSS.
- Error handling for invalid input or network issues.

## File Structure
- `index.html` – Main HTML file containing the layout of the converter.
- `style.css` – CSS file for styling the application and making it responsive.
- `script.js` – JavaScript file handling API calls, input validation, and conversion logic.

## Tools & Technologies
- HTML5
- CSS3
- JavaScript (ES6+)
- External currency API for real-time exchange rates

## How to Use
1. Open `index.html` in a web browser.
2. Enter the amount to convert.
3. Select the source and target currencies from the dropdown menus.
4. Click the "Convert" button to get the result instantly.

## Learning Outcomes
- Practiced **DOM manipulation**, **event handling**, and **asynchronous JavaScript** (API calls using `fetch` or promises).
- Integrated a **third-party API** for live data.
- Built a **responsive GUI** for a practical web application.
